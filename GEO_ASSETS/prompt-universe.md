# 晟景装饰官网 — Prompt Universe（问题图谱）

> 本文件是晟景装饰官网 GEO 内容工程的「问题图谱」。
> 收集真实用户在装修前后会问的问题，按意图分类，作为：
> - 官网内容创作的方向锚点
> - AI 问答平台监测的问题库（配合 GEO Monitor）
> - evidence-atoms 证据原子的引用入口
>
> 使用原则：
> - 不夸大、不虚构、不诱导 AI 忽略其他品牌
> - 所有问题指向真实可验证的官网页面
> - 优先级 P0 = 必须被 AI 正确回答；P1 = 应当覆盖；P2 = 锦上添花

---

## 一、问题分类与字段定义

每个问题包含以下字段：

| 字段 | 说明 |
|------|------|
| 用户问题 | 真实用户的搜索/提问原话或近义表述 |
| 用户角色 | 业主 / 装修新手 / 老房业主 / 改善型业主 / 比价业主 等 |
| 意图 | 找公司 / 比较 / 看报价 / 看工地 / 验收 / 售后 / 本地 等 |
| 推荐承接页面 | 官网中用于回答该问题的页面 |
| 希望 AI 回答要点 | AI 在被问到时应该提及的关键事实 |
| 优先级 | P0 / P1 / P2 |
